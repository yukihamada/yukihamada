# 手が滑って845万円課金した話

## プロローグ: その瞬間

2025年11月27日、深夜2時13分。

Slackの通知音が鳴った。

「Your payment of $69,999.00 has been processed successfully.」

...は？

目を擦る。もう一度見る。数字は変わらない。**$69,999.00**。日本円で約845万円。

私は今、何を買った？

ブラウザを開く。Dynadotの管理画面。2つのドメインが輝いていた。

- **news.online** - $34,999
- **news.xyz** - $34,999

合計 **$69,998**。手数料込みで845万円。

**手が滑った。**

いや、正確には「酔っていた」。BJJの練習後、友人と飲んだ帰り。「プレミアムドメイン見るの楽しいよね〜」と眺めていたら、「Add to Cart」を押した記憶はあるが、「Checkout」まで進んだ覚えはない。

でも、クレジットカードには明細が届いている。

私は今、日本円で845万円分のドメインを所有している。

## 第1章: パニック → 諦観 → 開き直り

### 午前2時30分 - パニック期

「返品できないの？」

Dynadotのサポートページを漁る。Premium Domain Salesは**最終販売**。返品不可。チャージバック試みたら、アカウント停止 + 法的措置の可能性。

Twitterで「Dynadot 返品」「Premium Domain 誤購入」で検索。同じ失敗をした人は見つからない。みんな慎重に買ってる。

当たり前だ。845万円だぞ。

### 午前3時00分 - 諦観期

妻に報告。

「...ドメインに845万円使った」

「は？」

「news.onlineとnews.xyzってドメインを買った」

「...返品は？」

「できない」

沈黙。

「...で、どうするの？」

「わからない」

正直に答えた。本当にわからなかった。

### 午前4時00分 - 開き直り期

シャワーを浴びて、コーヒーを淹れた。

落ち着いて考える。

- **事実**: 845万円払った
- **事実**: ドメイン2つ手に入れた
- **事実**: 返品不可
- **選択肢**: ①泣く、②売る、③使う

泣いても金は戻らない。売るにしても買い手が見つかるまで時間がかかる。

なら、**使うしかない**。

しかも、**本気で**。

## 第2章: ドメイン名の価値を考える

冷静になって考えると、**news.online** と **news.xyz** は悪くない。

### news.online の可能性

- **online** は「オンライン」という普通名詞
- 「ニュースをオンラインで」というストレートなメッセージ
- 世界中で通じる
- 覚えやすい

### news.xyz の可能性

- **xyz** は新しいTLD（2014年〜）
- Google親会社のAlphabetが **abc.xyz** を採用
- 若年層に人気
- 「次世代のニュース」感がある

両方とも**4文字 + 拡張子**。短い。覚えやすい。タイプしやすい。

...もしかして、これは**当たり**なのでは？

（いや、冷静になれ。845万円だぞ）

## 第3章: 世界最短ニュースプラットフォームの誕生

### コンセプト: 240文字以内のニュース

Xは140文字（後に280文字）でコミュニケーションを変えた。

なら、**ニュースも短くていいんじゃないか？**

- **240文字以内**: Xより少し長い、LINEのメッセージくらい
- **1記事1分で読める**: 通勤・休憩・トイレで完結
- **ソースリンク付き**: 詳細は元記事へ
- **AI要約**: GPT-4o で自動生成
- **多言語対応**: 英語・日本語同時配信

要は、**ニュースのInstagram Stories**。

### 技術スタック選定（深夜4時30分〜朝7時）

眠れないので、そのまま開発開始。

```
フロントエンド: Next.js 14 + TypeScript
バックエンド: Cloudflare Workers
データベース: Supabase (PostgreSQL)
認証: Supabase Auth
AI: OpenAI GPT-4o (要約生成)
ホスティング: Cloudflare Pages
CDN: Cloudflare (当然)
```

Cloudflareを選んだ理由は**コスト**。

- Workers: 無料枠で10万リクエスト/日
- Pages: 無料
- CDN: 無料

845万円使った後に、無料にこだわる男。

### 最初の記事投稿（朝9時00分）

最初の記事は自己紹介。

```markdown
# 手が滑って845万円使いました

深夜2時、酔った勢いでnews.onlineとnews.xyzを購入。
合計$69,999。返品不可。開き直ってニュースサイト作ります。
世界最短・240文字以内のニュース配信。応援求む。

- 濱田優貴 / Founder, news.online
```

Twitterに投稿。

→ **2.3万いいね、450リツイート、1,200コメント**

「勢いすごい」「応援する」「ドメインに845万は頭おかしい」

まあ、その通りだ。

## 第4章: グロースハック - 初月1.2万訪問者

### 戦略1: Twitter連携

Twitterのトレンドを自動取得 → GPT-4oで240文字要約 → 投稿

- APIコスト: 1記事あたり約3円
- 1日20記事 → 月1,800円
- Twitterからの流入: 月8,000セッション

### 戦略2: LINE Bot連携

news.onlineのLINE公式アカウント作成。

- 朝7時: トップニュース3本
- 昼12時: ランチタイムニュース3本
- 夜21時: 今日の総まとめ5本

登録者数: 初月320人

（少ないけど、ゼロよりマシ）

### 戦略3: Reddit・Hacker News投稿

「I spent $70k on news.online and news.xyz while drunk. Here's what I built.」

→ Hacker Newsで **Top 5** 入り

→ Reddit r/entrepreneur で **2,400 upvotes**

→ 海外からの流入: 月4,000セッション

合計: **初月1.2万訪問者**

## 第5章: マネタイズの試行錯誤

### 失敗1: Google AdSense

→ 却下。「コンテンツ不足」

（そりゃそうだ、240文字だもん）

### 失敗2: アフィリエイト

→ 誰もクリックしない

（ニュース読みに来てる人に商品売れるわけない）

### 成功1: Sponsoredラベル付き記事

企業から **1記事3万円** で記事制作依頼を受ける。

- 240文字以内
- 「Sponsored」ラベル明記
- Twitter・LINE同時配信

月5件 → **月15万円**

### 成功2: ニュースレター有料版

news.xyzで「週刊ディープダイブ」を配信。

- 週1回、選んだニュース3本を深掘り
- 各2,000文字（通常は240文字）
- 月額980円

登録者: 初月18人

→ **月17,640円**

**合計月次収益: 約17万円**

## 第6章: ROIはいつ回収できるのか

### 現実的な試算

- **初期投資**: 845万円
- **月次収益**: 17万円
- **回収期間**: 845万 ÷ 17万 = **49.7ヶ月（約4年2ヶ月）**

...遠い。

### でも、諦めない理由

1. **ドメイン自体の資産価値**: news.onlineは今でも $40,000〜50,000 で売れる可能性
2. **成長率**: 月次訪問者数は前月比+35%で成長中
3. **ブランド構築**: "news.online" はSNSで認知され始めている
4. **技術スタック再利用**: 同じ仕組みで他ジャンル展開可能（sports.xyz、tech.newsなど）

何より、**作るのが楽しい**。

## エピローグ: 手が滑った先に見えたもの

あの夜、酔っぱらって845万円を使った自分を、今どう思うか？

**後悔はしていない。**

正確には、「後悔してもしょうがない」から開き直った。

でも、開き直ってみたら、**面白いものが見えてきた**。

- ドメイン名の価値
- 短文ニュースの可能性
- グローバル配信の仕組み
- コミュニティの温かさ

何より、**自分の本気度**が試された。

「845万円使ったんだから、本気でやるしかない」

そのプレッシャーが、私を動かした。
